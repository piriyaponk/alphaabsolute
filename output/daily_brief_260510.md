- Phase 3 Inflection = ตัวเลขเริ่มพิสูจน์ story แต่ตลาดยังไม่เชื่อ 100% [BEST ENTRY]
- Phase 4 Recognition = ตลาดยอมรับแล้ว analyst upgrade เป็นแถว [ตลาดอยู่ตรงนี้]
- Phase 5 Consensus = ทุกคนเชื่อ ราคาขึ้นต่อแต่ช้าลง
- Phase 6 Euphoria = ทุกคนคุยเรื่องนี้ใน taxi [ระวัง]
- Phase 7 Distribution = สถาบันขายออก ข่าวดีแต่ราคาไม่ขึ้น

SMC Bullish Flow คืออะไร?

SMC = Smart Money Concept -- เงินใหญ่ทิ้งรอยไว้บน chart เสมอ:
- Stop Hunt = ราคาลงต่ำกว่า low เดิมแป๊บเดียวแล้วเด้งกลับ -> สถาบันซื้อตอน retail panic
- Break of Structure (BOS) = ราคาทำ Higher High ครั้งแรก -> trend เปลี่ยน
- Order Block Hold = pullback กลับมาแล้วไม่หลุด low เดิม -> สถาบัน defend ราคา

'SMC Bullish Flow Confirmed' = BOS ขึ้นแล้ว + order block เก่ายังเป็น support = ยังซื้ออยู่
