📐 PULSE Score — How We Rank Every Stock
🔢 ALPHA SCORE FORMULA
Score = (RS_Pct × 0.50) + (RS_Δ × 0.30) + (PULSE × 4) + (max(0, 25+FromHigh) × 0.3)

RS_Pct — Percentile rank 0–100 within PULSE universe (6M timeframe). Higher = stronger relative strength vs SPY.

RS_Δ — Rank acceleration: RS_Pct_1M minus RS_Pct_3M. Positive = stock is gaining speed vs universe. e.g. moved from 50th → 62nd = +12.

PULSE — 5-criteria checklist (0–5 passes): Stage 2 above 150D MA, above 200D MA, 150D > 200D, RS positive, near 52W high.

FromHigh — % from 52W high. Closer to ATH = higher contribution. Stocks >30% below ATH score 0 here.

Gates (must pass both):
• Stage 2 required (price > rising 30W MA)
• RS_Pct ≥ 50th (must be above median of universe)
📊 RS PERCENTILE RANK — How to Read
What it is: Where this stock ranks vs all stocks in PULSE universe (22 stocks) by 6M / 3M / 1M return vs SPY.

95th = top 5% — strongest RS in universe (Leader territory)
75th = top 25% — OVERWEIGHT zone
50th = median — minimum gate to be considered
25th = bottom quartile — avoid / watch only

Δ rank (rank change): 1M rank minus 3M rank.
+12 = stock moved up 12 positions in 1M vs where it was 3M ago → momentum accelerating
−8 = moved down 8 positions → momentum cooling → flag for review

Phase Changer threshold: |Δ rank| ≥ 8 positions triggers alert
🌀 NRGC — 7-Phase Narrative Cycle
Ph1 Neglect — RS <20th. Nobody cares. Vol dry. Dead chart.
Ph2 Accumulation — 20–35th. Smart money builds quietly. VCP forming.
Ph2→3 Inflection — 35–50th + RS accel. Story starts getting traction. BEST ASYMMETRIC ENTRY.
Ph3→4 Recognition — 50–60th + accel. Proof emerging. Momentum traders notice.
Ph4 Recognition — ≥75th. Consensus forming. Analysts upgrade. Institutional FOMO.
Ph5–6 Consensus/Euphoria — ≥90th near ATH. Everyone owns it. Risk of reversal.
Ph7 Distribution — Wyckoff UTAD/SOW. Good news = price doesn't move. Exit signal.
🎯 WYCKOFF × WEINSTEIN GATE
Mandatory double gate — both must pass for any BUY:

Weinstein Stage 2: Price > rising 30W MA. MA trending up. RS positive. If Stage 3/4 → automatic fail.

Wyckoff Phase:
• Accumulation C/D (Spring, LPS, SOS) → GREEN gate ✅
• Mark-Up early (CHoCH, BUEC) → GREEN gate ✅
• Distribution (UTAD, SOW, LPSY) → RED gate 🔴 — no buy regardless of fundamentals

Entry signals (WHY this price):
• VCP Contraction: vol dry-up <0.6x avg, tight range (<8% depth)
• Wyckoff LPS: pullback after SOS on lower vol = re-entry
• Spring: price dips below support + volume spike + snap back
• SOS Breakout: vol >1.5x avg + price bar closes near high
