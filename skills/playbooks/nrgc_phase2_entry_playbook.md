# NRGC Phase 2 Entry Playbook
*Setup for early institutional discovery — highest R/R zone*

Load this playbook when a stock is transitioning from NRGC Phase 1 (Neglect) to Phase 2 (Early Acceleration).

---

## What is NRGC Phase 2?

Phase 2 = **Early Acceleration** — the moment when:
- Revenue/EPS starts growing (QoQ turns positive)
- Guidance improves for the first time
- First breakout from a base forms
- Smart money begins quiet accumulation

This is the HIGHEST R/R entry point — before the crowd discovers it.

---

## Phase 2 Detection Signals (EMLS Boost Criteria)

**Fundamental inflection (most important):**
- [ ] EPS turned positive QoQ (from negative or flat)
- [ ] Revenue acceleration: QoQ positive for 2+ consecutive quarters
- [ ] Gross margin expansion beginning
- [ ] Guidance raised or first positive guidance issued

**Earnings tone signals:**
- [ ] Tone shift from neutral/negative to positive (earnings_tone.py)
- [ ] Management using inflection language: "turning the corner", "accelerating", "strong pipeline"
- [ ] First upward EPS estimate revision from analysts

**Price / RS signals:**
- [ ] RS percentile crossing above 50th from below
- [ ] First base forming (Base 0 = first base from Phase 1 bottom)
- [ ] Price reclaiming 150D MA for the first time
- [ ] Volume pattern changing: accumulation days outpacing distribution

**Narrative signals:**
- [ ] Keyword frequency for the stock's theme accelerating WoW
- [ ] First institutional 13F filing showing new position
- [ ] Short interest declining (covering begins)

---

## Phase 2 Entry Checklist

**Before entering:**
- [ ] Revenue QoQ positive for at least 1 quarter (Phase 2 confirmed)
- [ ] Base 0 or Base 1 ONLY — not more than 2 bases from Phase 1 bottom
- [ ] RS line turning up — does not need to be top quartile yet
- [ ] NOT in Stage 3 or 4 price structure
- [ ] Narrative exists but NOT crowded yet (low analyst coverage, few retail mentions)

**Sizing rules for Phase 2:**
```
Base 0 max: 5% (hypergrowth rule applies)
Base 1 max: 8%
Upgrade to full size only when: RS > 72nd pct + Stage 2 confirmed + 3+ quarters acceleration
```

---

## Phase 2 vs Phase 3 — Know the Difference

| Signal | Phase 2 (Entry Zone) | Phase 3 (Still OK) | Phase 4 (Avoid) |
|--------|---------------------|---------------------|-----------------|
| Revenue growth | Just turned positive | Accelerating 3+ qtrs | Peak growth known |
| RS percentile | 40-65th | 65-85th | 85-95th (crowded) |
| Analyst coverage | Low (0-5 analysts) | Growing (5-15) | High (15+) |
| Retail mention | Rare | Growing | Everywhere |
| Base count | 0-1 | 1-2 | 3+ (late) |
| Price vs 52W low | +30-80% | +80-150% | +150%+ (extended) |

## Exit / Upgrade Triggers

**Upgrade to full position when:**
- RS crosses 72nd percentile (all timeframes)
- 3rd consecutive revenue acceleration quarter confirmed
- Stage 2 fully confirmed (Weinstein)
- EMLS reaches 80+

**Exit Phase 2 early if:**
- Revenue QoQ goes negative again (thesis broken)
- Tone deteriorates back to negative/guarded
- Base fails (price breaks below base low on volume)
- Short interest RISING despite price up (smart money shorting rally)

- **MU (Memory)**: Phase 2 entry when DRAM pricing inflected (Q2 2023)
- **NVDA**: Phase 2 to 3 when datacenter revenue first exploded (Q1 2023)
- **RKLB**: Phase 2 on first profitable launch economics confirmed

Pattern: First 2 quarters of acceleration = Phase 2 window. By quarter 3 confirmed = already Phase 3.
